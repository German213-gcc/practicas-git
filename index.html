Guía de Configuración del Proyecto

Sigue estos pasos para asegurar la persistencia de los datos:

  1. Configurar el archivo db-pgsql.php con las credenciales de PostgreSQL.
  2. Verificar que el servidor local (XAMPP/WAMP) esté activo.
  3. Ejecutar el formulario para insertar datos a través de usuarios.php.
  4. Sincronizar los cambios con el comando git push origin main.
