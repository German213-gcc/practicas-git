Práctica 2: Estructura y Marcado de Texto

Introducción al HTML

Según el manual de la unidad, el HTML se utiliza para dar estructura a las páginas web. No basta con escribir el contenido, debemos usar etiquetas para que el navegador entienda qué es cada elemento.

Uso de Marcado Semántico

En esta práctica estamos aprendiendo a resaltar información importante. Por ejemplo, podemos poner palabras en negrita para darles importancia o usar cursivas para dar énfasis a ciertos términos técnicos de programación web.

Notas adicionales

El modelo Cliente-Servidor es fundamental. En esta práctica, mi navegador actúa como el cliente que interpreta este código HTML que he estructurado.
